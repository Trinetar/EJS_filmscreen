-----------------------------------------------------------------------
Welcome to the filmmybox app,
An electronJS app.
-----------------------------------------------------------------------
Poster of the movies are stored inside 'images' directory with specific name in jpeg format.

Under src/renderer.js file description of each image is stored and can be modified by adding changing or deleting.

-----------------------------------------------------------------------
nodeJS and npm must be installed 

-----------------------------------------------------------------------
run by 'npx electron .' 
Click on poster to read its description.
